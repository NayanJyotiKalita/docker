# Managing Docker Swarm Services

Objective:

Managing Docker Swarm Services

TASK - 1:

Activate the docker swarm service in the given environment.

TASK - 2:

Deploy a new service called webapp using the nginx image with two replicas.

TASK - 3:

Scale the service up to three replicas.

TASK - 4:

Save the output of docker service ls command and docker service ps webapp command to a file named swarm_service_output.txt in the /home/user/ directory.

---

# CODE

```bash
docker swarm init
docker service create –name webapp –replicas 2 nginx
docker service scale webapp=3
docker service ls
docker service ps webapp
docker service ls > /home/user/swarm_service_output.txt
docker service ps webapp > /home/user/swarm_service_output.txt
```
