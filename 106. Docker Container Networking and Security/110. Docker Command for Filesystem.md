# Docker Command for Filesystem

Which command runs a container with a read-only filesystem?

Choose the correct answer from below:

docker run -d --cap-drop ALL --cap-add NETBINDSERVICE nginx

docker run -d --read-only nginx

docker run -d --security-opt seccomp=/path/to/seccomp/profile.json nginx

docker run -d --security-opt label:type:containerruntimet nginx
