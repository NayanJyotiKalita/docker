# Configuring Docker Bridge Network

## Objective:

Configuring Docker Bridge Network

TASK - 1:
Create a custom Docker bridge network named my_bridge_network.

TASK - 2:
Run a new container called web_container using the nginx image and connect it to the my_bridge_network.

TASK - 3:
Verify that the container is connected to the bridge network by inspecting the network and container.
