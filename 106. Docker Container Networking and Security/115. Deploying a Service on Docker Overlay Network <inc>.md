# Deploying a Service on Docker Overlay Network

Objective:

Deploying a Service on Docker Overlay Network

TASK - 1:

Initialize a Docker Swarm.

TASK - 2:

Create an overlay network named my_overlay_network.

TASK - 3:

Deploy a new service called overlay_service using the nginx image on the my_overlay_network.

TASK - 4:

Scale the service to 3 replicas.

TASK - 5:

Verify the deployment and scaling by listing the services and tasks.
