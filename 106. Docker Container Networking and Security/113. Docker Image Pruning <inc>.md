# Docker Image Pruning

## Objective
Learn how to effectively manage Docker storage by pruning unused images, containers, and volumes. This task will help you reclaim disk space, maintain a clean development environment, and optimize Docker performance.

## Instructions

  - Identify unused Docker images, containers, and volumes on the host.

  - Remove or prune the unused Docker objects safely.

  - Verify that the unused resources have been removed and disk space has been reclaimed.

---

# COMMANDS
```shell

# List the available images
docker images

# List any containers using the images
docker ps

# Stop containers
docker stop <container_id>

# Remove containers
docker rm <container_id>

# Remove all the images
docker image prune -a

# Now list the images to verify whether they are removed or not
docker images

```
